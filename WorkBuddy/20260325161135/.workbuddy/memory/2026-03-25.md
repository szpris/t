## NasiTracker 项目创建

完成了一个完整的 Android 位置追踪系统的全部代码，包含：

### Android App（tracker-app/）
- **语言**：Kotlin，最低 SDK 26，目标 SDK 34
- **核心功能**：每60秒或位移超200米上报一次位置，FusedLocationProviderClient
- **后台运行**：ForegroundService + START_STICKY，支持开机自启
- **离线缓存**：网络失败时保存到本地 SQLite，恢复后自动补传
- **UI**：启动/停止/刷新按钮，当前位置显示，最近20笔记录列表（含序号和时间）
- **调试界面**：连续点击状态栏5次触发，支持测试上传当前位置，支持手动输入经纬度上传
- **认证**：API_KEY = "NasiTracker2026SecretKey#X9mZ"（与服务端一致）
- **服务器地址**：http://track.nasi.cn:55000

### 服务端（tracker-server/）
- **框架**：Python + Flask + SQLite3
- **端口**：55000
- **认证**：API Key（上报接口），用户名/密码（Web界面）
- **Web管理**：登录认证，按时间范围查询，分页显示
- **数据导出**：CSV、高德地图HTML、百度地图HTML、谷歌地图HTML
- **系统服务**：提供 systemd 配置文件

### 编译方式
- 推荐使用 GitHub Actions（.github/workflows/build.yml 已内置）
- 将 tracker-app 目录 push 到 GitHub 仓库，Actions 自动编译产出 APK

### 重要配置
- API_KEY 在 Constants.kt 和 server.py 两处需保持一致
- Web 登录：admin / NasiAdmin2026!（服务端 server.py 中修改）
